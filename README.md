Contibutors : Carolina Fereira, Lahouari Belabbes, Robin d'Ercole

Le morpion:

Jeu légendaire s'il en est...

Le but du jeu est d'aligner 3 symboles. 3 directions possibles, horizontale, verticale et diagonale. 

On a crée différentes classes : 

- BoardCase qui crée les cases
- Board qui dispose les cases pour créer le " plateau "
- Player pour associer les joueurs à leur symboles respectifs
- Game qui permet d'alterner les tours de jeu entre player 1 et player 2
